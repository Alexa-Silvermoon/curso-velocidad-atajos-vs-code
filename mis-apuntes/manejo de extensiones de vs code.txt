
paste JSON as Code:
practicar copiando al pokemon en https://pokeapi.co/
https://www.udemy.com/course/vscode-mejora-tu-velocidad-para-codificar/learn/lecture/13968338#notes

1 - copiar la informacion json en la pagina api

2 - ir a vs code y escribir el comando Ctrl + Shift + P y seleccionar la opcion
    paste JSON as Code

3- en el cuadrito que aparece escribir el nombre que queremos darle a lo que habiamos copiado y
   luego dar enter, si todo sale bien se habra pegado de forma bonita lo que
   habiamos copiado en el paso 1

Bookmarks:
es como cuando uno pone un lazo en un libro para saber donde quedo la lectura, los bookmarks funcionan
igual, para poner un bookmarks se escribe el comando Ctrl + Alt + K y se activa una marca azul

Live Server:
sirve para ejecutar rapidamente un archivo html y correrlo como si estuviera ya en internet

para ello solamente oprimir el botn que dice Go Live y ya veremos nuestro archivo html corriendo
en chrome, para detener el servidor dar click en el mismo boton que ahora dira Port: 5500
